Welcome to Better Design

This site explores the fundamental concepts of graphic design, including typography, color theory, and layout.

Good design is not just about making things look good; it is about effective communication.

Key Terms

Kerning
The spacing between individual characters in a word.
Leading
The vertical distance between lines of text.
Hex Code
A six-digit code used to represent colors on the web.
